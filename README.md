#アプリ名

ArtField

#サイト概要

個人製作した作品を拡散できるサイト

#サイトテーマ

芸術と人を繋ぐ

#テーマを選んだ理由

身近に個人で美術品を製作してる方がいて、芸術に特化した個人で作った作品を広める事ができるアプリがあればいいなと感じた。

#ターゲットユーザ

芸術分野を専攻してる学生、個人製作してる人

#主な利用シーン

自分の製作物をたくさんの方に見てもらい時。
